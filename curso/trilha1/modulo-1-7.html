üîÑ Mesmo pipeline, parametros diferentes

O pipeline GIPM e o mesmo em todos os niveis - o que muda sao os parametros de configuracao. Isso permite evoluir sem reescrever.

üîß Exemplo: Parametro "persistence_level"

Exploratorio

persistence_level: "none"
log_inputs: false
log_outputs: false
track_costs: false

Operacional

persistence_level: "basic"
log_inputs: false
log_outputs: true
track_costs: true

Institucional

persistence_level: "full"
log_inputs: true
log_outputs: true
track_costs: true

üí° Vantagem

Voce pode comecar no nivel exploratorio e subir para institucional mudando apenas configuracao, sem refatorar a arquitetura.
